# ☕ Coffee Shop Sales Analysis (MS Excel)

A data analytics project built using **Microsoft Excel** to analyze coffee shop sales, customer footfall, and product performance through an interactive dashboard.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Excel Functions

## Project Highlights
- Analyzed total sales, footfall, average bill, and order metrics
- Identified peak ordering hours and weekday sales trends
- Performed category-wise and size-wise sales distribution analysis
- Compared sales and footfall across multiple store locations
- Identified top-performing products based on sales

## Dashboard Insights
- Morning hours record the highest order quantity
- Coffee category contributes the largest share of total sales
- Regular and Large sizes dominate order volume
- Weekdays show consistent customer footfall
- Store-wise sales performance varies by location

## Files
- `Coffee Shop Sales.xlsx` – Dataset and Excel dashboard  
- `Coffee Shop Sales Analysis.pdf` – Exported dashboard report

## Outcome
Transformed raw sales data into actionable business insights using MS Excel dashboards.
